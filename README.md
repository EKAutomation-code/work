# ПлюсБаллы Fullstack

Новая версия проекта под стек:

- Frontend: React + Vite
- Backend: FastAPI
- Database: PostgreSQL
- Files: S3-compatible storage, локально MinIO, на проде Яндекс Object Storage
- Payments: точки интеграции под ЮKassa / CloudPayments
- Deploy: Docker + VPS

## Запуск локально

1. Скопируйте `.env.example` в `.env`.
2. В `.env` задайте `JWT_SECRET` и `ADMIN_PASSWORD`.
3. Запустите:

```bash
docker compose up --build
```

После запуска:

- сайт: http://localhost:5173
- API: http://localhost:8000/docs
- MinIO console: http://localhost:9001

## Что уже заложено

- регистрация ученика и репетитора;
- подтверждение email-кодом, в dev-режиме код печатается в логах backend;
- вход по email, паролю и одноразовому коду;
- админ-панель с одобрением аккаунтов;
- KYC обязателен только для репетиторов перед апрувом;
- репетитор после апрува публикует предмет ЕГЭ, цену и описание;
- ученик выбирает репетитора и прикрепляется к нему;
- репетитор создает домашние задания;
- ученик загружает фото/PDF через S3-presigned upload;
- учитель выставляет оценку от 1 до 5;
- лидерборд считается по средней оценке за проверенные ДЗ;
- платежный слой подготовлен отдельным роутером, секреты остаются только на backend.

## Перед продакшеном

- подключить реальную почту вместо вывода кодов в лог;
- заменить MinIO-настройки на Яндекс Object Storage;
- реализовать проверку webhook-подписей ЮKassa/CloudPayments;
- добавить миграции Alembic вместо `Base.metadata.create_all`;
- включить HTTPS, домен и reverse proxy на VPS;
- хранить секреты только в `.env` на сервере, не коммитить `.env`.
